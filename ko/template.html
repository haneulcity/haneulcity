2013년, 맵 개발의 시작

종이상자는 2013년부터 탈출맵 'THE LAVA'를 시작으로 도시맵 개발에 몰두했습니다. 일반적인 도시맵이 Minecraft : Java Edition을 기반으로 하는 것과 달리, 무한 맵이 아니라는 한계에도 꾸준한 개발로 SandCity와 ArtCity를 개발하여 중간 개발 판을 내놓기도 했습니다.

그러나 맵 개발은 생각보다 쉽지 않았습니다. 포켓에디션도 0.9.0을 기점으로 무한 맵이 도입되면서 한계를 벗어나 더 넓은 꿈을 펼칠 수 있게 되었으나, 기존 맵을 무한 맵으로 전환하는 것은 이질적일 뿐더러 정상적으로 변환되지도 않아 제대로된 건축이 어려웠습니다. 그래서 SandCity와 ArtCity를 모두 계승하는 작품, 하늘마을 맵이 시작되었습니다.
