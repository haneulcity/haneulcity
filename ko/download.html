Github Unstable Build

하늘마을 맵은 공식적으로 Github에서 맵 파일을 관리합니다. 매 변경사항이 있을 때마다 새로운 일간 빌드가 올라오며, 제작자의 사정에 따라 간혹 한 번에 변경점을 반영한 빌드가 올라오기도 합니다. 과거 버전도 찾아 내려받을 수 있는 장점이 있으나 변경될 때마다 반영되지는 않으므로 실시간 성격을 띄지는 않습니다.

내려받기 과거 판본 찾아보기

무엇이 바뀌었는지 알고 싶으신가요? 과거 판본 찾아보기를 누르신 후, 각 빌드 제목 옆에 붙은 ... 버튼을 눌러보세요.

Box Real-Time Build(일시 중단)

일시 중단된 빌드입니다.

Box 클라우드와 실시간으로 동기화되는 저장소로, Minecraft로 자동 저장될 때마다 판올림되는 체계입니다. Github Unstable Build보다 더 즉각적으로 반영되나 파일이 올라가는 도중 내려받는 등의 상황이 발생할 수 있는 만큼 주의가 필요합니다.
