Contribute

Help HaneulCity with your hand.

1: Suggest, report bugs to Issue Tracker

HaneulCity doesn't use WorldEdit plugin except some exceptions, so some holes and spelling issues can existed. If you find bugs or spelling issues from this map, please write a report to Issue Tracker of this project. You can report it with Github Account. No Account? Don't Worry. There are ways to report with E-mail and Twitter.

Report to Issue Tracker Send an e-mail Mention with Twitter

2: Donate buildings @ Minecraft Map

If you want to donate buildings that you built, send map file, contact with us and decide where to built.

To send map file, please use e-mail , not pull-request. It is not possible to build your buildings on where buildings exist.

3: Translate

You can traslate signs in HaneulCity map with forking this project, or translate this website. Please contact with Issue Tracker, e-mail, or Twitter before start traslate.

4: Donate

Currently, you can donate via PayPal.Me

For South Korea, see this pageto donate.

To Register contributor list, please mail me via "Send an e-mail" button with capture image that shows you donated to this project.
