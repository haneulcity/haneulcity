For Bedrock Edition Only

Various buildings, unique subways

In HaneulCity map, all buildings are different. For example, if there is a library per village, they have different design by town.

It is same for more than 5 Lines of subway. Different station, line color, and others.

Not for external, for internal

Did you ever discoraged because a nice building didn't have any interior? Don't worry. HaneulCity map provides internal design per building.

This rule will kept until the project dead.
